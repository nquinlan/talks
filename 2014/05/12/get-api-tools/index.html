Tools Covered

The tools covered in the talk are as follows:

Web APIs

  • Postman - REST Client as a Chrome Extension
  • Runscope - API Debugging Tools
  • Unirest - Common REST Interface API Library

Webhooks

  • Request Bin - Storage and inspector for HTTP Requests
  • ngrok - Inspectable tunnels to your machine
  • Reflector.io - Super nifty webhook tools

Nick Quinlan is a Developer Evangelist for SendGrid where he works to makes developers lives easier.
